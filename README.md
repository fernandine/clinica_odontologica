
# Clínica Odontológica

Projeto teste de uma clínica odontológica usando Java e Angular 2+.

## Tecnologias usadas
- Angular 16.1.0
- Spring Boot 3+
- Angular Material
- Jasper
- ModelMapper

Banco de Dados: H2 / MYSQL





## Execução

Para executar o projeto rode:

```bash
  ng s --port 8081
```
